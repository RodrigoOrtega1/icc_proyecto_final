## Formato
Formato correcto de archivo para sobreescribir metadata, si se quieren ver ejemplos consultar la carpeta archivos ejemplo:
* llave : valor (Una llave separada por dos puntos y un valor

Campos EXIF que edita el programa:
* LensModel
* DateTimeOriginal
* DateTimeDigitized
* UserComment
* OwnerName
* CameraOwnerName
* BodySerialNumber
* Latitude
* Longitude

> :warning: **Latitude y longitude van en pares!:** Es decir no puedes cambiar la latitud o longitud individualmente
